# advent-of-code-typescript

A project for solving AOC challenges in typescript. Each day's puzzle is in a seperate folder. Change the import path in src/index.ts to match the desired solution file.

example.txt contains the given test-case examples for a puzzle.
input.txt contains my personalized puzzle input.

## Notes

I will use the following format for solution files where appropriate:
    
- imports and setup    
- read input
- call to the solving function, logging the return
- solving function(s)

I will use the same solution file for both parts of each puzzle (unless that gets messy).
