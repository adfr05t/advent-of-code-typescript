# 2017 Day 2: Corruption Checksum

## Puzzle
https://adventofcode.com/2017/day/2

## Part 1
- Aim: 
    - Find the difference between the highest and lowest numbers in each row.
    - Sum those differences.
        
## Part 2
- Aim: 
    - Find the only two numbers in each line where one divides into the other giving a whole number as the result. 
    - Sum the results of each of these divisions.
