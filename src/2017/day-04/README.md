# 2017 Day 4: High-Entropy Passphrases

## Puzzle

https://adventofcode.com/2017/day/4

## Part 1

- Aim:
  - Find the number of valid pass phrases in the input. A valid pass phrase is one that contains no duplicated words.

## Part 2

- Aim:
  - As part 1, but now a pass phrase is invalid if it contains words that are an anagram of each other eg. abdf ghr fbad is invalid.
